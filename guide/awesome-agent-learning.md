## Foundational Courses

- **[Advanced Large Language Model Agents](https://llmagents-learning.org/)** -- Graduate-level course exploring LLM-powered agent design and deployment. Covers LLM foundations, reasoning, tool use, multi-agent collaboration, and applications with guest lectures from leading researchers.

- **[Agentic AI and AI Agents: A Primer for Leaders](https://www.coursera.org/learn/agentic-ai)** -- Concise course for non-technical executives and product managers covering agent fundamentals and no-code implementation using custom GPTs.

- **[AI Agents Masterclass](https://github.com/coleam00/ai-agents-masterclass)** -- Beginner-friendly episodic series with code walkthroughs. Covers LangChain, LangGraph, RAG techniques, and n8n workflow agents with accompanying code.

- **[Hugging Face's AI Agents Course](https://huggingface.co/learn/agents-course/)** -- Hands-on program from foundational concepts to deployment. Covers implementation using smolagents, LlamaIndex, and LangGraph with assignments and final project.

- **[Learn AI Agents Handbook](https://www.learn-agents.diy/)** -- Interactive handbook and roadmap for building autonomous AI agents.

- **[Microsoft's AI Agents for Beginners](https://github.com/microsoft/ai-agents-for-beginners)** -- Comprehensive open-source course with 11 lessons covering tool integration, RAG, design patterns, multi-agent systems, and production deployment. Includes materials, code samples, and videos.

- **[Multi AI Agent Systems with crewAI](https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/)** -- Beginner-friendly course on building and deploying agents using CrewAI. Covers tools management, memory organization, error handling, and agent cooperation with business process examples.

## Conceptual Guides

- **[A Practical Guide to Building Agents (OpenAI)](https://cdn.openai.com/business-guides-and-resources/a-practical-guide-to-building-agents.pdf)** -- Step-by-step playbook offering best practices for designing autonomous agents, covering problem identification, architecture, and success measurement.

- **[A Survey on Large Language Model based Autonomous Agents](https://arxiv.org/abs/2308.11432)** -- Academic survey systematically examining LLM-based autonomous agent construction, applications, and evaluation with discussion of evaluation strategies and future research directions.

- **[Building effective agents (Anthropic)](https://www.anthropic.com/engineering/building-effective-agents)** -- In-depth guide with practical strategies for LLM-based agent design. Emphasizes starting with simple composable patterns like prompt chaining before advancing to autonomous tool-using agents.

- **[LLM Powered Autonomous Agents (Lilian Weng)](https://lilianweng.github.io/posts/2023-06-23-agent/)** -- Comprehensive blog post exploring autonomous LLM agent architecture, covering theory and implementation of agent components.

- **[What Are AI Agents?](https://medium.com/codex/what-are-ai-agents-your-step-by-step-guide-to-build-your-own-df54193e2de3)** -- Introduction to AI agents with step-by-step building guidance, focusing on business architecture with implementation left to readers.

## Framework Tutorials

- **[AutoGPT Guide](https://docs.agpt.co/platform/getting-started/)** -- Step-by-step tutorial for local AutoGPT platform setup covering backend and frontend configurations for building and managing agents.

- **[CrewAI Quickstart Guide](https://docs.crewai.com/quickstart)** -- Hands-on tutorial for building multi-agent systems using CrewAI, demonstrating role definition, task assignment, and collaboration orchestration.

- **[Haystack Tutorials](https://haystack.deepset.ai/tutorials)** -- Collection of tutorials covering building, evaluating, and deploying agentic pipelines.

- **[Hugging Face smolagents Tutorials](https://huggingface.co/docs/smolagents/tutorials/building_good_agents)** -- Examples outlining best practices for building agents using the smolagents framework.

- **[LangChain: Build an Agent](https://python.langchain.com/docs/tutorials/agents/)** -- Hands-on guide demonstrating how to build AI assistants capable of tool use with minimal setup.

- **[LlamaIndex Learning](https://docs.llamaindex.ai/en/stable/understanding/)** -- Guide for building agentic applications using LlamaIndex, covering LLM basics, agent construction, workflow design, and RAG implementation.

- **[Microsoft AutoGen AgentChat Quickstart & Tutorial](https://microsoft.github.io/autogen/stable/user-guide/agentchat-user-guide/quickstart.html)** -- Quick tutorial for building AI assistants using Microsoft's AutoGen AgentChat framework.

- **[OpenAI Agents SDK Examples](https://openai.github.io/openai-agents-python/examples/)** -- Comprehensive collection of practical examples organized by agent patterns, tool integration, model providers, and real-world applications covering design patterns and multi-turn orchestration.

- **[OpenAI Assistants API](https://platform.openai.com/docs/assistants/quickstart)** -- Step-by-step tutorial on building AI assistants using OpenAI's Assistants API, covering assistant creation, tool integration, and conversation management.

## Evaluation Benchmarks

- **[AgentBench](https://github.com/THUDM/AgentBench)** -- Systematic multi-domain benchmark for measuring LLMs in autonomous agent roles spanning 8 tasks including OS management, SQL operations, web browsing, games, and puzzles with a leaderboard.

- **[BrowseComp](https://openai.com/index/browsecomp/)** -- Benchmark featuring 1200+ human-crafted questions assessing agent ability to find hard-to-locate information across the web.

- **[GAIA](https://huggingface.co/spaces/gaia-benchmark/leaderboard)** -- 450-question benchmark testing agent reasoning, web search, tool-use, and multimodal understanding through short-answer tasks across three difficulty levels.

- **[OSWorld](https://github.com/xlang-ai/OSWorld)** -- Benchmark for evaluating multimodal agents on 350+ real web and desktop tasks including UI/GUI manipulations, graded by execution traces.

- **[SWE-bench](https://github.com/SWE-bench/SWE-bench)** -- Large-scale benchmark with 2k+ real-world GitHub issues from Python repositories where agents must generate patches and pass tests in Docker environments. Includes Lite, Verified, and Multimodal versions.

- **[ToolBench](https://github.com/sambanova/toolbench)** -- 8-task benchmark requiring agents to call real APIs from various web services including weather, spreadsheets, shopping, and reservations.

- **[WebArena](https://webarena.dev/)** -- Realistic self-hosted benchmark environment with 4 web applications (e-commerce, forums, CMS, code) featuring 800+ long-horizon tasks.
